# Zod Exercises

A mini project with 10 exercises to learn Zod. Each exercise has a starter file and a corresponding test.

## Setup

```bash
npm install
```

## Run tests

```bash
npm test
```

## How to work

- Implement each exercise in `src/exercises/`.
- Run tests in `tests/` to check your work.
- Each exercise file includes a TODO block describing the goal.

## Exercises

1. `src/exercises/01_primitives.ts`: Basic primitives.
2. `src/exercises/02_refinements.ts`: Refinements and custom error messages.
3. `src/exercises/03_nested_objects.ts`: Nested object schemas.
4. `src/exercises/04_optional_defaults.ts`: Optional fields and defaults.
5. `src/exercises/05_arrays_tuples.ts`: Arrays and tuples.
6. `src/exercises/06_unions_enums.ts`: Unions and enums.
7. `src/exercises/07_discriminated_union.ts`: Discriminated unions.
8. `src/exercises/08_transforms.ts`: Transforms and coercion.
9. `src/exercises/09_partial_deep_partial.ts`: Partial and deep partial.
10. `src/exercises/10_composition.ts`: Schema composition.
